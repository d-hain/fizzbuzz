## How to run

Install any of the C++ compilers.\
Here I'll use gcc.\
Install gcc from [gcc.gnu.org](https://gcc.gnu.org).\
Go into the [cpp folder](/cpp) and run the following command:
```shell
gcc fizzbuzz.cpp
```
And to run it on windows run:
```shell
a.exe
```
